🎮 3D Puzzle Game

Rust + wgpu で開発されたクロスプラットフォームゲーム

⏳ 開発中： WASM 版は現在準備中です。以下の方法でプレイできます。

🎯 ゲーム内容

3D 空間に配置された立方体を正しい位置に移動するパズルゲーム。光る線エフェクトとリアルタイム物理演算が特徴です。

🚀 はじめ方

1. ネイティブ版（最初に試す）

git clone https://github.com/oosawak/rustgames.git cd rustgames cargo run --release -p native_app

2. WASM 版（ブラウザ）

wasm-pack build wasm_app --target web --release cd wasm_app/pkg python -m http.server 8000 # ブラウザで http://localhost:8000 を開く

3. Unity プラグイン

cargo build --release -p ffi_bridge # DLL/so/dylib を Unity/Assets/Plugins にコピー
